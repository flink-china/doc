Next Generation of Big Data Processing Engine

Apache Flink is a framework and distributed processing engine for stateful computations over unbounded and bounded data streams. Flink has been designed to run in all common cluster environments, perform computations at in-memory speed and at any scale.

Apache Flink Features

Apache Flink provides powerful Big Data processing engine and easy-to-use programming interface.

Unified Engine for batch and stream process

A unified runtime and SQL layer for both stream and batch process. Apache Flink provides low latency and high throughput computation ability with a more powerful SQL engine.

Deploy Applications Anywhere

Apache Flink runs on Hadoop, Apache Mesos, Kubernetes, standalone, or in the cloud. It can access diverse data sources.

Excellent Performance

The batch and stream processing performance are excellent.

Run Applications at any Scale

Apache Flink is designed to run stateful streaming applications at any scale. Applications are parallelized into possibly thousands of tasks that are distributed and concurrently executed in a cluster.
